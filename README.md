# Sistema de Gestão de Estoque – Sushi

##  Descrição
Este projeto é um sistema web desenvolvido em **Python com Django** para gestão de estoque de um restaurante de sushi.

O sistema permite controlar produtos, categorias, fornecedores e movimentações de estoque, facilitando o acompanhamento e organização dos itens.

---

## Funcionalidades

- Cadastro de produtos
- Gestão de categorias
- Cadastro de fornecedores
- Controle de inventário
- Registro de movimentações (entrada/saída)
- Relatórios de estoque

---

## Tecnologias utilizadas

- Python
- Django
- SQLite (banco de dados)
- HTML / CSS (templates)

---

## Nota sobre o banco de dados

Este projeto inclui uma base de dados SQLite com dados de teste,
portanto não é necessário executar migrações nem criar um superutilizador.
