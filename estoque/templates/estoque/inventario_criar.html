O que acontece ao criar?
  • Um inventário em aberto é criado com a data de hoje.
  • Todos os produtos ativos são adicionados automaticamente com o saldo atual do sistema.
  • O teu papel é corrigir o saldo físico contado fisicamente no armazém.
  • As divergências são calculadas automaticamente ao guardar.
Cancelar
Como funciona o inventário

1. Criar — o sistema regista o saldo que tem em base de dados para cada produto.

2. Contar — a equipa conta fisicamente cada produto no armazém e insere a quantidade real.

3. Divergências — o sistema calcula automaticamente a diferença entre o saldo do sistema e o saldo físico contado.

4. Concluir — após confirmar todos os valores, o inventário é fechado e fica no histórico.
